# xuan-react-scripts

自己定制的 react-scripts   
兼容 create-react-app 2.0.4 版本

# 使用指南

## 基于`xuan-react-scripts`创建项目

```bash
create-react-app [project-name] --scripts-version @react-script/normal
```

## 样式

- 支持 sass scss stylus

- 支持 css-module

- 支持 variables mixins 自动注入
